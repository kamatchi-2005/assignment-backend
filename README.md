 📌 Assignment Tracker – Backend

This is the Spring Boot backend for the Student Assignment Tracker System.
It manages student assignments, project details, status tracking, and data storage.

🚀 Live Backend URL (Render – Free Plan)

👉 https://assignment-backend-ram9.onrender.com/api/assignments

⚠️ Important Note (Render Free Plan):

This backend is hosted on Render Free Plan.

Backend may go to sleep when idle

When opening the frontend, backend may take 30–60 seconds to wake up

Open backend URL once before using the frontend
